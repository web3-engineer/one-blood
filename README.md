# Project Name

Jogo de plataforma 2D com elementos 3D, foco em exploração, combate responsivo e atmosfera sombria.

## Stack
- Engine: Godot 4
- Linguagem: GDScript
- Plataforma inicial: PC

## Estrutura
- `docs/` documentação do projeto
- `game/` projeto principal
- `.github/workflows/` automações de CI

## Objetivo inicial
Construir um vertical slice jogável com:
- movimentação base
- combate básico
- 1 inimigo
- 1 área explorável
- checkpoint
